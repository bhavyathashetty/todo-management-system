# 🚀 Todo Management System

A modern and responsive Todo Management application built to help users efficiently organize, manage, and track daily tasks.

This project demonstrates full-stack development concepts including task CRUD operations, frontend-backend integration, API handling, and responsive UI design.

---

## ✨ Features

- ✅ Add new tasks
- 📝 Update existing tasks
- ❌ Delete tasks
- 📋 View all tasks
- 🔄 Real-time task management
- 📱 Responsive user interface
- ⚡ Fast and clean user experience

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone git@github.com:bhavyathashetty/todo-management-system.git
```

### 2️⃣ Navigate into the Project

```bash
cd todo-management-system
```

### 3️⃣ Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

---

## ▶️ Run the Application

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```
